# 🌌 Cosmic Theme — React Native Template

> Atmospheric, time-aware, poetic theme system for my React Native apps.

**Status:** 🚧 Setup phase — Faz 0 başlangıç

## What is this?

Bu repo bir **template**. GitHub'da "Use this template" tıklayarak yeni uygulamalarımda Cosmic atmosfer + 12 karakter sistemini hazır olarak alıyorum.

## Quick Start (Yeni Uygulama Yapacağın Zaman)

```bash
# 1. GitHub'da "Use this template" tıkla
# 2. Yeni repo oluştur (örn: my-new-app)

# 3. Klonla ve çalıştır:
git clone https://github.com/YOUR-USERNAME/my-new-app.git
cd my-new-app
npm install
npx expo start
```

Bitti! Cosmic atmosfer + 12 karakter hazır.
